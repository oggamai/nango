## API gotchas

- The Vitally Authorization header format is basic auth, using the API Key as the username.
- There are different domains (not just subdomains) for US and EU datacenters: `https://{subdomain}.rest.vitally.io` (US) vs `https://rest.vitally-eu.io` (EU)
- Vitally enforces rate limits of 1000 requests per minute using a sliding window. For more details, check [the REST API Overview](https://docs.vitally.io/en/articles/9880649-rest-api-overview).
